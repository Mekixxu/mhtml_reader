# Update App Version Spec

## Why
The user wants to set the initial version of the app to `0.0.1`.

## What Changes
- Change `versionName` in `app/build.gradle.kts` from `0.1.0` to `0.0.1`.
- Keep `versionCode` as `1`.

## Impact
- Affected code: `app/build.gradle.kts`

## ADDED Requirements
None.

## MODIFIED Requirements
### Requirement: App Version
The app version name SHALL be `0.0.1`.
